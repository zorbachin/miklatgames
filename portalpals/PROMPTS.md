# 🎨 PORTAL PALS — image & video prompt pack v1

Copy-paste prompts for **ChatGPT (GPT-image)** and **Google Flow
(Imagen for stills / Veo for clips)**. Everything derives from
`WORLD.md` — if a prompt drifts from the bible, fix the prompt.

## How to use (read once)

1. **Always start with the MASTER STYLE block** — paste it verbatim at the
   top of every prompt. Consistency comes from repeating the exact same
   style language, not from luck.
2. **Order matters.** Generate each character's *reference sheet* first.
   Then, for every later image, attach/reference that sheet ("use the
   attached character reference; keep design identical") — both ChatGPT
   and Flow support image references, and this is the single biggest
   consistency lever.
3. **One new thing per image.** One character per sheet; introduce the
   LASS alone before putting Pals in it; combine only after each element
   has a locked reference.
4. **Aspect ratios:** reference sheets & sprites 1:1 · environments and
   key art 16:9 · poster 9:16 · banner 21:9 (or 16:9 crop).
5. **In Flow**, put the SHOT/AUDIO lines in the prompt exactly as written
   — Veo uses them. For stills in Flow, use the same text minus SHOT/AUDIO.

## 0 · MASTER STYLE (paste at the top of every prompt)

```
STYLE: Vibrant 3D animated feature film style (modern theatrical CG family
movie). Soft rounded shapes, chunky silhouettes readable at thumbnail size,
big expressive eyes, clean PBR materials on robots, subtle subsurface skin.
Palette: warm desert golds and teal skies; brand accent is glowing portal
orange (#FF7A1A); villain tech is the ONLY violet light. Sunny key light
with soft bounce, rim-lit action, portals act as practical orange light
sources. Low, kid-height camera angles, wide lens adventure scale.
FOR KIDS: zero gore, zero realistic firearms — all gadgets are glowing
light-energy tools. No text, no watermark, no logo unless specified.
```
